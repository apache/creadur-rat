# Creadur RAT Migration Guide - Release 0.18

Release 0.18 is an intermediate release as we are working on big architectural changes within the 1.0.0 feature development.

## Maven

* [RAT-508](https://issues.apache.org/jira/browse/RAT-508) reduced the logging output of exclusion parsers and the exclusion engine. If you want to see these messages run Maven with the ```-X``` option or set logging to debug.
